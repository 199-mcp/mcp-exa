Exa MCP Server

A Model Context Protocol server for AI-powered web search

Available Tools

web_search
Searches the web in real-time. Returns: page content, titles, URLs. Use when: need current information beyond training data.
query: Search query (e.g., 'OpenAI GPT-5 release')
numResults: Number of results (1-20, default: 5)
liveCrawl: 'always' | 'preferred' | 'fallback'
url_content
Extracts full content from specific URLs. Returns: complete page text, metadata. Use when: have exact URL to analyze.
url: URL to extract content from
maxCharacters: Maximum characters (1000-10000, default: 3000)
liveCrawl: 'always' | 'preferred' | 'fallback'
academic_search
Searches academic papers and research. Returns: paper abstracts, citations, authors. Use when: need peer-reviewed sources.
query: Academic search query
numResults: Number of papers (1-20, default: 5)
company_search
Searches company information and news. Returns: business data, financials, recent news. Use when: researching businesses or organizations.
companyName: Company name (e.g., 'OpenAI', 'Tesla Inc')
numResults: Number of results (1-20, default: 5)
competitor_search
Finds business competitors. Returns: similar companies, market analysis. Use when: asked 'who competes with X' or competitive analysis.
companyName: Company to analyze
industry: Industry sector (optional)
numResults: Number of competitors (1-20, default: 5)
linkedin_search
Searches LinkedIn profiles and companies. Returns: professional profiles, company pages. Use when: researching people or professional networks.
query: Search query
searchType: 'profiles' | 'companies' | 'all'
numResults: Number of results (1-20, default: 5)
wikipedia_search
Searches Wikipedia encyclopedia. Returns: article summaries, factual content. Use when: need encyclopedic or reference information.
query: Search query
numResults: Number of articles (1-20, default: 5)
github_search
Searches GitHub repositories and code. Returns: repos, code snippets, READMEs. Use when: looking for code examples or open source projects.
query: Search query
searchType: 'repositories' | 'code' | 'users' | 'all'
numResults: Number of results (1-20, default: 5)

Usage

SSE Endpoint: https://exa.atp.dev/api/mcp?exaApiKey=YOUR_API_KEY
